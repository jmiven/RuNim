# RuNim

### Compiles and Runs a Nim File

`runim main.nim`

### To Also Recompile On File Change

`runim main.nim -w`

Or

`runim main.nim --watch`

### Silent Mode `--verbosity:0 --hints:off`

`runim main.nim -s`

Or

`runim main.nim --watch --silent`

### Installation

`nimble install https://github.com/s0kil/RuNim`
